LinksApp is a customizable tool that allows the creation of a grouping page of links and information. 

# installation 

• cloning or repository :
```
git clone https://github.com/sebastianjnuwu/LinksApp
```
• installing all dependencies:
```
yarn install
```
• starting project:
```
yarn start
```
